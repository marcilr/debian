1. Backup the system.
Mostly backed up to revision control.

2. Update existing packages and reboot the Debian 10 system.
$ sudo aptitude update
[sudo] password for marcilr: ***
...
$ 

$ sudo aptitude upgrade
No packages will be installed, upgraded, or removed.
0 packages upgraded, 0 newly installed, 0 to remove and 0 not upgraded.
Need to get 0 B of archives. After unpacking 0 B will be used.

$

Looks good!

3. Edit the file /etc/apt/sources.list using a text editor and 
replace each instance of buster with bullseye. Next find the 
security line, replace keyword buster/updates with bullseye-security.

$ cd /etc/apt

Backup sources.list configured for Debian 10 Buster:
$ sudo cp sources.list sources.list-BUSTER-20221228

$ md5sum sources.list  sources.list-BUSTER-20221228
cb8511a3d193e471dd08fa6526f13267  sources.list
cb8511a3d193e471dd08fa6526f13267  sources.list-BUSTER-20221228

Update buster with bullseye in /etc/apt/sources.list using sed:
$ sudo sed -i 's/buster/bullseye/g' /etc/apt/sources.list
$ grep buster /etc/apt/sources.list
$

$ grep bullseye /etc/apt/sources.list
# deb cdrom:[Official Debian GNU/Linux Live 10.1.0 xfce 2019-09-07T11:18]/ bullseye main
# deb cdrom:[Official Debian GNU/Linux Live 10.1.0 xfce 2019-09-07T11:18]/ bullseye main

$

Determine line of buster/updates to update with bullseye-security:

$ grep -ni buster/updates sources.list-BUSTER-20221228 
10:deb http://security.debian.org/debian-security buster/updates main
11:deb-src http://security.debian.org/debian-security buster/updates main
$

Update buster/updates with bullseye-security in /etc/apt/sources.list:
$ sudo nano +10 /etc/apt/sources.list
deb http://security.debian.org/debian-security bullseye/updates main
===> deb http://security.debian.org/debian-security bullseye-security main

$ sudo nano +11 /etc/apt/sources.list
deb-src http://deb.debian.org/debian/ bullseye-updates main
===> deb-src http://deb.debian.org/debian/ bullseye-security main

The /etc/apt/sources.list update from buster to bullseye looks good.
