Updated debian with:

$ sudo aptitude update
$ sudo aptitude upgrade
$ sudo /shutdown -r now
$ cat /etc/debian_version 11.6

Interestingly the newly updated Debian 11.5 booted just fine. Previously I've observed on Debian updates that that system had to be completely shutdown and started manually for the reboot to occur. Looks like this issue has been resolved.
